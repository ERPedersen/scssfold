# Row

| Class         | Result                                          |
| ------------- | ----------------------------------------------- |
| .row          | A row element with negative margins.            |

## Install separately

```scss
@import '~scssfold/src/modules/layout/row';
```

## Customizing

Override the following variables before including the module.

```scss
$scssfold-grid-gutter: 20px;
```
